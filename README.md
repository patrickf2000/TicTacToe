# TicTacToe
This is a tic-tac-toe game written using MonoGame. The has been tested and developed on Windows 10 (using VS 2017) and on Linux (using MonoDevelop). It is licensed under the MIT license.

### Features
This supports one and two player modes. Winning combinations are highlighted in green. In the event of a draw, all letters turn blue.

### Dependencies
All that is needed is MonoGame. MonoGame can be installed on any OS and can work as a plugin in Visual Studio and MonoDevelop. On Linux, you will need Mono.

### Building
Simply open the solution in a C# IDE.

### Branches
The master branch is the stable branch. The dev branch is the current development branch. If trying this out for the first time, build from the master branch.
